# Character Creation Overview

This document walks you through the full character creation process for *Inner Tepenia*.

## Step-by-Step Creation Flow

### 1. MACHINE Stats (1–10 scale)
Distribute your starting points across the 7 core attributes.

| Letter | Stat            | Core Focus                                                                 | Fallout Equivalent      |
|--------|-----------------|----------------------------------------------------------------------------|-------------------------|
| **M**  | **Might**       | Raw physical strength and power                                            | Strength                |
| **A**  | **Agility**     | Speed, reflexes, coordination, and fine motor control                      | Agility                 |
| **C**  | **Calculation** | Logical processing, computation, optimization, and direct problem-solving  | Intelligence + Luck     |
| **H**  | **Humanity**    | Emotional intelligence, empathy, social awareness, and connection          | Perception + Charisma   |
| **I**  | **Investigation** | Pattern recognition, information gathering, analysis, and deduction      | — (New)                 |
| **N**  | **Nerve**       | Mental resilience, willpower, and influence over companions                | New stat                |
| **E**  | **Engine**      | Operational endurance and how long you can function before burning out     | Endurance               |

### 2. Calculate Skill Points per Level
**Formula**: `floor( (Calculation ÷ 2) + (Nerve ÷ 2) )`

This determines how many skill points you gain each level (typically 3–8).

### 3. Choose Tag Skills (3 Tags)
After determining your skill points per level, select **3 skills** to Tag.

- Each Tagged skill immediately receives:  
  **(Skill points per level) + 5**

**Example**: If you have 6 skill points per level, each Tagged skill starts with **+11 points**.

### 5. Starting Skills
- Base starting skills are low (usually 5–25 depending on MACHINE stats).
- Apply your **Tag Skill** bonuses.
- Governing MACHINE stats provide small passive bonuses during play.

### 6. Perks & Progression
- Perks are earned roughly every 3 levels.
- Many perks have requirements that are much easier if you Tagged the relevant skill.
- Hidden paths (Independent Lattice, etc.) are heavily gated by specific stat + skill + perk combinations.

## Recommended Starting Builds

**Tech / Hidden Paths Build**  
High Calculation + Engine + Investigation → Tag: Decentralized Systems Design, Jury-Rigging, Ripple Reading

**Social / Diplomat Build**  
High Humanity + Nerve → Tag: Diplomatic Negotiation, Empathy Protocols, Faction Reputation Management

**Frontier / Survivalist Build**  
High Engine + Might + Agility → Tag: Frontier Survival, Environmental Exploitation, Stealth & Infiltration

**Crisis Exploiter / Narrative Build**  
High Investigation + Calculation + Humanity → Tag: Deception & Narrative Crafting, Data Archaeology, Ripple Reading
