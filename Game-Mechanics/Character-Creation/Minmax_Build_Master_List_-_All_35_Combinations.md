# Complete Minmax Build Master List - All 35 Combinations

**Document Status**: Complete Draft  
**Total Combinations**: C(7,3) = 35  
**Format**: Each build lists a **High (Max)** version (three stats 8+) and a **Low (Min)** version (three stats 3 or below).

**Stats Key**:  
**M** = Might **A** = Agility **C** = Calculation **H** = Humanity **I** = Investigation **N** = Nerve **E** = Engine
